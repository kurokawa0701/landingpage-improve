GASコードを貼り付ける

① 通知先メールアドレスを設定する（3行目）

コード内の NOTIFY_EMAIL の値を通知を受け取りたいメールアドレスに変更してください。

// ======================================================
// 設定：通知メールの送信先アドレス
// ======================================================
const NOTIFY_EMAIL = 'your-email@example.com'; // ← ここを変更

// ======================================================
// POST リクエスト受信（フォーム送信時に呼ばれる）
// ======================================================
function doPost(e) {
  try {
    const params = e.parameter;
    const type   = params.type || 'contact';  // "download" or contact types

    if (type === 'download') {
      handleDownload(params);
    } else {
      handleContact(params);
    }

    return ContentService
      .createTextOutput(JSON.stringify({ status: 'ok' }))
      .setMimeType(ContentService.MimeType.JSON);

  } catch (err) {
    Logger.log(err);
    return ContentService
      .createTextOutput(JSON.stringify({ status: 'error', message: err.message }))
      .setMimeType(ContentService.MimeType.JSON);
  }
}

// ======================================================
// 資料請求フォーム処理
// ======================================================
function handleDownload(p) {
  const ss    = SpreadsheetApp.getActiveSpreadsheet();
  const sheet = ss.getSheetByName('資料請求');
  const now   = new Date();

  // ヘッダーが無ければ追加
  if (sheet.getLastRow() === 0) {
    sheet.appendRow(['受信日時', '会社名', 'お名前', 'メールアドレス']);
    sheet.getRange(1, 1, 1, 4).setFontWeight('bold').setBackground('#163A5C').setFontColor('#ffffff');
  }

  sheet.appendRow([
    Utilities.formatDate(now, 'Asia/Tokyo', 'yyyy/MM/dd HH:mm:ss'),
    p.company || '',
    p.name    || '',
    p.email   || ''
  ]);

  // 通知メール送信
  const subject = '【資料請求】' + (p.company || '') + ' ' + (p.name || '') + ' 様';
  const body = [
    '資料請求がありました。',
    '',
    '■ 会社名：' + (p.company || '未入力'),
    '■ お名前：' + (p.name    || '未入力'),
    '■ メール：' + (p.email   || '未入力'),
    '',
    '受信日時：' + Utilities.formatDate(now, 'Asia/Tokyo', 'yyyy/MM/dd HH:mm:ss')
  ].join('\n');

  MailApp.sendEmail(NOTIFY_EMAIL, subject, body);
}

// ======================================================
// お問い合わせフォーム処理
// ======================================================
function handleContact(p) {
  const ss    = SpreadsheetApp.getActiveSpreadsheet();
  const sheet = ss.getSheetByName('お問い合わせ');
  const now   = new Date();

  // ヘッダーが無ければ追加
  if (sheet.getLastRow() === 0) {
    sheet.appendRow(['受信日時', '会社名', 'お名前', 'メール', '電話番号', '種別', 'お問い合わせ内容']);
    sheet.getRange(1, 1, 1, 7).setFontWeight('bold').setBackground('#163A5C').setFontColor('#ffffff');
  }

  sheet.appendRow([
    Utilities.formatDate(now, 'Asia/Tokyo', 'yyyy/MM/dd HH:mm:ss'),
    p.company || '',
    p.name    || '',
    p.email   || '',
    p.phone   || '',
    p.type    || '',
    p.message || ''
  ]);

  // 通知メール送信
  const subject = '【お問い合わせ】' + (p.company || '') + ' ' + (p.name || '') + ' 様';
  const body = [
    'お問い合わせがありました。',
    '',
    '■ 会社名　：' + (p.company || '未入力'),
    '■ お名前　：' + (p.name    || '未入力'),
    '■ メール　：' + (p.email   || '未入力'),
    '■ 電話番号：' + (p.phone   || '未入力'),
    '■ 種別　　：' + (p.type    || '未入力'),
    '■ 内容　　：',
    (p.message || '未入力'),
    '',
    '受信日時：' + Utilities.formatDate(now, 'Asia/Tokyo', 'yyyy/MM/dd HH:mm:ss')
  ].join('\n');

  MailApp.sendEmail(NOTIFY_EMAIL, subject, body);
}
⚠️ 4行目の your-email@example.com を必ず変更してください。ここに設定したアドレスに通知メールが届きます。
4
スクリプトを保存してデプロイする
ウェブアプリとして公開します

保存

  1. エディタ上部のフロッピーアイコン（💾）または Ctrl + S（Mac: ⌘ + S）で保存する

デプロイ

  1. 右上の 「デプロイ」→「新しいデプロイ」 をクリックする
  2. 「種類の選択」の歯車アイコン⚙️をクリックし、「ウェブアプリ」 を選択する
  3. 以下の通りに設定する
設定項目設定値
説明LP フォーム受信 v1
次のユーザーとして実行自分（your@gmail.com）
アクセスできるユーザー全員（Googleアカウント不要）← 必須
  1. 「デプロイ」ボタンをクリックする
  2. 「アクセスを承認」 が表示されたら → 自分のGoogleアカウントを選択 → 「詳細」→「安全でないページへ移動」→「許可」をクリック
  3. 「デプロイが完了しました」の画面で 「ウェブアプリのURL」 をコピーする
🔴 「アクセスできるユーザー」が 「全員」 でないとLPからのフォーム送信が受信できません。必ず「全員（Googleアカウントなし）」または「全員」に設定してください。

コピーしたURLはこのような形式になります：

https://script.google.com/macros/s/AKfycbXXXXXXXXXX/exec
5
main.js のURLを更新する
LPとGASを接続します

ses-sales-partner-site/main.js の1行目を、コピーしたURLに書き換えます。

// main.js の1行目
const GAS_URL = 'ここにコピーしたURLを貼り付ける';
💡 すでに前回セッションでURLが設定済みの場合は、新しいデプロイのURLに必ず上書きしてください。デプロイのたびに新しいURLが発行されます。
6
動作テストをする
送信 → スプレッドシート確認 → メール確認
  1. LPをブラウザで開き、「無料相談する」ボタンをクリックしてモーダルを開く
  2. テスト用の情報（会社名・名前・メール等）を入力して「送信する」をクリックする
  3. 「送信が完了しました」の画面が表示されることを確認する
  4. Googleスプレッドシートの お問い合わせ シートに行が追加されていることを確認する
  5. 指定のメールアドレスに通知メールが届いていることを確認する
  6. 同様に 「資料をメールで受け取る」 フォームもテストする
⚠️ フォームは mode: 'no-cors' で送信するため、ブラウザ側では成功・失敗の判定ができません。スプレッドシートへの書き込みを必ず目視確認してください。
7
（任意）コードを修正した場合の再デプロイ
コードを変更したら必ず新バージョンをデプロイしてください
  1. スクリプトエディタで修正後、Ctrl + S で保存する
  2. 「デプロイ」→ 「デプロイを管理」 をクリックする
  3. 既存のデプロイの右にある鉛筆アイコン（✏️）をクリックする
  4. 「バージョン」を 「新バージョン」 に変更し、「デプロイ」をクリックする
💡 「デプロイを管理」から更新すると、URLは変わりません。main.jsの書き換えは不要です。
✅

設定完了！

これでLPのフォームから送信されたデータが、スプレッドシートに自動記録され、
指定のメールアドレスに通知が届くようになります。
